Blackjack Game

Text based rendition of the popular card game, 'Blackjack'.

Project Summary

This text based 'Blackjack' game allows user to pick an amount to deposity into their vitrual casino account. Ask user to deposit a valid amount to bet on each hand. Allows user to play the Computer ('The Dealer') in a game of blackjack (21). Records round number, and total earnings and balance.

class Card():

def __init__(self,suit,rank):
	
	self.suit = suit
	self.rank = rank
	self.value = values[rank]

def __str__(self):
	return(self.rank + " of " + self.suit)

class Deck():

def __init__(self):

	for suit in suits:
		for rank in ranks:  
			newcard = Card(suit,rank) 
			all_cards.append(newcard)
		
def deal_one(self):
	
	return all_cards.pop(0)

def shuffle(self):
	
	random.shuffle(all_cards)
