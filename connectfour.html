Connect Four Game

Modern rendition of the classic board game Connect Four.

Project Summary

This project is a clean representaion of a Connect Four game, allowing 2 users to and locally play against each other. The game then request that the users select a column to drop their specific colored chip, and once four chips are placed in a row the game concludes. This was my first JavaScript centered project, and was developed in conjunction with Jquery.

Developed with: JavaScript (Jquery), CSS, HTML
